psas-player

Easy streaming from any computer with an internet connection.

What do you need to download?

  • vmlinuz-X.X.XX and initramfs-psas-player-X.X.XX.DATE.cpio.gz, the kernel and initramfs image; can be used to PXEboot or with SYSLINUX for USB flash drives
  • psas-player.VERSION.DATE.tar - a zipfile containing the kernel and initramfs images
  • psas-player.VERSION.DATE.iso - an ISO image you burn with CD writing software

Usage

Burn the ISO, create a bootable USB flash drive with syslinux or PXEboot the kernel and initramfs image, then boot up the system. There's a cheat code menu on the ISO image, just hit F2 after the system boots the ISO image.

Enjoy!
